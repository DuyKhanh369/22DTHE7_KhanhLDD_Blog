## Giới thiệu

Java là một ngôn ngữ lập trình hướng đối tượng mạnh mẽ, được phát triển bởi Sun Microsystems (nay thuộc Oracle).

## Đặc điểm chính

- **Hướng đối tượng**: Java hoàn toàn là ngôn ngữ OOP
- **Độc lập nền tảng**: "Write Once, Run Anywhere"
- **Bảo mật cao**: Có nhiều cơ chế bảo mật tích hợp
- **Đa luồng**: Hỗ trợ lập trình đa luồng

## Hello World trong Java

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

## Kết luận

Java là ngôn ngữ lập trình quan trọng cần học cho các lập trình viên.
