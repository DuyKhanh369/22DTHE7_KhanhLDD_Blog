Giới thiệu về JavaScript

25/12/2025 JavaScript Cơ bản

JavaScript là gì?

JavaScript là ngôn ngữ lập trình phổ biến nhất cho phát triển web, chạy trên trình duyệt và server (Node.js).

Đặc điểm

  • Ngôn ngữ thông dịch: Không cần biên dịch
  • Động: Kiểu dữ liệu được xác định lúc runtime
  • Đa paradigm: Hỗ trợ OOP, functional, imperative

Hello World

console.log("Hello, World!");

Biến trong JavaScript

// ES5
var name = "JavaScript";

// ES6+
let age = 30;
const PI = 3.14;

Arrow Function

// Traditional function
function add(a, b) {
    return a + b;
}

// Arrow function
const add = (a, b) => a + b;

Kết luận

JavaScript là ngôn ngữ cần thiết cho mọi web developer.
